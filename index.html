카드뉴스 생성 설정


          

            
📌 1번에서 만든 보험 제안서 HTML 파일과 디자인 참고 이미지를 ChatGPT에 함께 업로드합니다. 이 프롬프트는 ChatGPT 이미지 생성 전용입니다.

          


          
          

            
            

              
              
            

            

              🆓 무료버전 특징: '추론' 모델 선택 불가. 모든 카드가 하나의 이미지에 나란히 생성됩니다.
→ 스크린샷 후 이미지 편집으로 각 카드를 잘라 PNG/JPG로 저장해서 사용합니다.
→ 4~6장 생성을 권장합니다.
            

            

              ⭐ 유료버전 특징: 모델 선택 시 '추론(o1/o3)' 선택 → 각 카드가 개별 이미지로 생성됩니다.
→ 각 이미지 우측 다운로드 버튼으로 개별 저장 가능합니다.
→ 장수 제한 없이 원하는 만큼 생성 가능합니다.

🖼️ Gemini AI 직접 이미지 생성

            

              
🆓 Gemini API로 카드뉴스 이미지를 앱에서 바로 생성합니다.

              무료 API는 분당 생성 횟수 제한이 있으며 이미지를 순차 생성합니다.

              

                ⭐ 유료 결제 API + Imagen 3 활성화시 더 선명하고 고품질의 이미지를 생성할 수 있습니다. (API 설정 → Imagen 3 체크)
              

            

            

              
              
            

            

              
            

            

              
카드뉴스 이미지를 생성하고 있습니다...

사용 방법 — 워크플로우


          

            
⚠️ ChatGPT 바로가기 → 에서 진행합니다. Claude AI가 아닌 ChatGPT 이미지 생성 기능을 사용합니다.

          


          
          

            
            
          


          
          

            

              
1

              

                
디자인 참고 이미지 준비

                
핀터레스트 '카드뉴스' 검색 바로가기 →
마음에 드는 디자인 이미지 캡처/저장 (1~3장 추천)

              

            

            

              
2

              

                
이 앱에서 프롬프트 복사

                
설정 완료 → '카드뉴스 프롬프트 생성하기' → 📋 프롬프트 복사하기 클릭

              

            

            

              
3

              

                
ChatGPT 접속 (무료 계정)

                
chat.openai.com 바로가기 → 로그인 → 새 대화(+) 시작

              

            

            

              
4

              

                
📎 파일 2가지 첨부 + 프롬프트 붙여넣기

                

                  
① 📋 입력창에 복사한 프롬프트 붙여넣기 (Ctrl+V)

                  
② 📄 1번에서 만든 보험 제안서 HTML 첨부

                  
③ 🖼️ Pinterest 디자인 참고 이미지 첨부

                

              

            

            

              
5

              

                
전송 → 결과 수신

                
전송 → 1장의 이미지에 4~6장 카드가 나란히 생성

              

            

            

              ✂️ 무료버전 이미지 자르기 방법

              생성된 이미지 스크린샷 저장

              → 윈도우: 캡처 도구 또는 그림판으로 카드별 잘라내기

              → 맥: 미리보기 앱에서 자르기

              → 각 카드를 PNG 또는 JPG로 저장 후 업로드
